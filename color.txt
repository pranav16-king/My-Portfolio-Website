⭐ 🔵 1. PARTICIPATION THEME — “SkyFresh Minimal”
Purpose:

For participation, webinars, hackathons, quizzes, campus events, online sessions.

Color Psychology:

Blue → trust, clarity, learning

Sky blue → freshness, simplicity

Visual Styling:

Top Bar Gradient: from-[#0EA5E9] to-[#0284C7]

Background: from-white via-[#F0F8FF] to-[#E6F4FF]

Border: border-sky-200

Text Highlights: #0284C7

Shadow: shadow-2xl (soft, clean)

Image Frame: subtle shimmer + soft sky ring

Buttons: sky-blue border → hover sky fill

Animation / Effects:

✔ Fade-in
✔ Image shimmer hover
✔ Clean minimal academic aesthetic

Usage Examples:

SkillEcted webinars,
Hack4Us,
TATA Crucible,
HP Power Lab,
SkillEcted Python/Cyber/Webinar certificates,
Any general participation.

🔷 AUTHORIZATION SECTION (for Participation Theme)

This section is SAME for all themes, only colors change.

Purpose:

To maintain formal, clean certification credibility.

Structure:
<div class="flex flex-col text-xs text-gray-700 leading-relaxed">
  <span><strong>Status:</strong> PARTICIPATION</span>
  <span><strong>Signed By:</strong> Issuer / Program Coordinator</span>
  <span><strong>Issued By:</strong> Organization / Institute</span>
  <span><strong>Reference Code:</strong> EVENT-CODE</span>
  <span><strong>Date Issued:</strong> DD Month YYYY</span>
</div>

Style Rules:

✔ Always text-xs, text-gray-700
✔ Same sequence for every certificate
✔ Never use color inside signature block (always gray)
✔ 4–5 lines max
✔ Keeps uniformity on your entire website
